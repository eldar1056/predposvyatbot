
- <b>/start</b> - Присылает приветственное сообщение.
- <b>/help</b> - Показывает описание возможных команд.
- <b>/status</b> - Отображает строку для каждой группы, в которой написано где она находится в данный момент, ее заработанное количество баллов и количество пройденных этапов.
- <b>/stages</b> - Отображает название каждого этапа с никами проводящих.
- <b>/scores</b> - Для каждой группы отображает количество полученных баллов и разбиение на этапы, на которых баллы были получены.
