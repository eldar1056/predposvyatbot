
- /start - Присылает приветственное сообщение
- /help - Показывает описание возможных команд
- /status - Отображает строку для каждой группы, в которой написано где она находится в данный момент, ее заработанное количество баллов и количество пройденных этапов
- /stages - Отображает название каждого этапа с никами проводящих
- /scores - Для каждой группы отображает количество полученных баллов и разбиение на этапы, на которых баллы были получены

Если у вас есть вопросы/предложения, можете написать:вопрос [ваш вопрос]
Возожно вы получите ответ
